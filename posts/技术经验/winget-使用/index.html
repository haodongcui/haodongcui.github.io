Winget 使用

作者: HaoDong Cui 发表于: 2024-10-29 分类: 计算机大杂烩 标签: 计算机, Win11

切换源

以管理员权限运行终端， 切换 winget 源为 USTC 镜像

winget source remove winget
winget source add winget https://mirrors.ustc.edu.cn/winget-source

重置为微软官方源：

winget source reset winget

搜索软件

winget search <软件名称/关键词>

安装软件

winget install <软件名称>

卸载软件

winget uninstall <软件名称>

更新软件

winget upgrade <软件名称>

查看已安装软件

winget list
