wsl2 使用

作者: HaoDong Cui 发表于: 2024-08-21 分类: 计算机大杂烩 标签: 计算机, wsl2, Linux, Ubuntu

操作系统：Windows11

在任务栏搜索栏中，搜索“功能”，打开“启用或关闭 Windows 功能”，开启“适用于 Windows 的 Linux 子系统”和“虚拟化平台”，开启后会提示重启电脑，重启即可。

安装

打开终端，安装（默认是 Ubuntu）

wsl --install

Linux 输入密码时，是不显示的，输入正确按回车即可

查看已安装的子系统

查看已安装的子系统

wsl --list -v

可简写为

wsl -l -v

查看可安装的发行版

查看可安装的发行版

wsl --list --online

可简写为

wsl -l --online

下载指定的发行版（比如 kali-linux 是可安装的发行版，现安装它作演示）

wsl --install kali-linux

卸载

卸载 kali-linux

wsl --unregister kali-linux

查看现有子系统

wsl -l -v

备份与恢复

导出 Ubuntu 子系统至目前所在目录的 ubuntu.tar

wsl --export Ubuntu ubuntu.tar

从 ubuntu.tar 的路径导入，并命名为 Ubuntu2

wsl --import Ubuntu2 E:\Desktop\ubuntu.tar

查看现有子系统

wsl -l -v

退出子系统

进入子系统后，退出回到命令行

exit

关闭子系统

关闭所有子系统

wsl --shutdown

查看现有子系统状态

wsl -l -v
