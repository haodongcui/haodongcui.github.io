自动换源 -chsrc ​

chsrc: 全平台命令行换源工具 chsrc，目标支持 Linux (包括麒麟、openEuler、deepin 等), Windows, macOS, BSD 等尽可能多的操作系统，龙芯、飞 (gitee.com)

安装 (x64)

bash
curl -L https://gitee.com/RubyMetric/chsrc/releases/download/pre/chsrc-x64-linux -o chsrc; chmod +x ./chsrc

运行换源命令

bash
sudo ~/chsrc set ubuntu
