Ubuntu й…ҚзҪ® nvm й•ңеғҸ вҖӢ

й…ҚзҪ®й•ңеғҸеҸӮиҖғпјҡдҪҝз”Ёnvmз®ЎзҗҶnodeеӨҡзүҲжң¬пјҲе®үиЈ…гҖҒеҚёиҪҪnvmпјҢй…ҚзҪ®зҺҜеўғеҸҳйҮҸпјҢжӣҙжҚўnpmж·ҳе®қй•ңеғҸпјү_node зүҲжң¬з®ЎзҗҶ-CSDNеҚҡе®ў

зӣҙжҺҘи®ҫзҪ®й•ңеғҸжәҗ

shell
npm config set registry https://registry.npmmirror.com

дҝ®ж”№й•ңеғҸжәҗ

shell
# 1. жё…з©әзј“еӯҳ
npm cache clean --force
# 2. еҲҮжҚўж–°жәҗ
npm config set registry https://registry.npmmirror.com
# 3. жЈҖжҹҘжҳҜеҗҰи®ҫзҪ®жҲҗеҠҹ
npm config get registry

nvm еҹәжң¬ж“ҚдҪң вҖӢ

shell
# жҹҘзңӢе·Іе®үиЈ…зҡ„nodejsзүҲжң¬(дёӨжқЎе‘Ҫд»ӨйғҪеҸҜ)

nvm list installed

# жҳҫзӨәжүҖжңүеҸҜд»ҘдёӢиҪҪзҡ„зүҲжң¬

# е®үиЈ…жҢҮе®ҡnodejsзүҲжң¬

# е®үиЈ…жңҖж–°зҡ„еҸҜз”ЁзүҲжң¬

# еҲҮжҚўдёәжҢҮе®ҡзүҲжң¬

# еҚёиҪҪжҢҮе®ҡзүҲжң¬
