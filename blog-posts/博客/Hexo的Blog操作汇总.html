搭建流程

安装hexo

npm install -g hexo-cli

初始化hexo

hexo init [RootFolderName]

RootFolderName可加可不加

安装package.json中的依赖

npm install

写作流程

写文章

hexo new PageName

PageName为文章名称，允许中文，不包含后缀名.md

新Page新建后，Front Matter会进行初始化，自动设置title为PageName，可修改；此外，date会自动设置为当前时间，可修改。

Hexo本地预览页面

清除缓存并生成页面

在blog文件夹下，右键点击Open Git Bash Here，输入以下命令

hexo clean
hexo generate
hexo serve

或简写如下

hexo clean
hexo g
hexo s

预览

预览地址：

http://localhost/4000/

推送到GitHub

hexo clean
hexo generate
hexo deploy

或简写如下

hexo clean
hexo g
hexo d

注意

  • 在搭建Hexo博客时，若在cmd中使用bash命令，如果出现错误因为在此系统上禁止运行脚本，可在设置->系统->开发者选项->PowerShell中勾选更改执行策略
