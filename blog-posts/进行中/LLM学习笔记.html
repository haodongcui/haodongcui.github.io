1-1 自然语言处理基础 ​

Word Representation 词表示 ​

One-Hot Representation
利用单位向量表示单词，空间维度为为单词列表长度
劣势：正交系，相似度始终为 0

Co-Occurrence Counts (同现计数)
represnet word by contexts
劣势：单词出现频次少，则不准确

Word Embedding
分布式的词表示，建立一个低维的稠密向量空间
代表性工作：Word2Vec

Language Model 语言模型 ​

根据前文预测下一句话

考虑两个概率
Conditional probability 根据前文预测的下一句话的概率
Joint probability 预测的内容成一句话的概率

N-gram
基于 Markov 假设，e.g.
Trigram (N=3)，只考虑邻近的两个词，如
P(learn|Never,too,late,to)≈P(learn|late,to)
劣势：仅考虑词组，句子在含义上的相似度无法表示等

Network Language Model
分布式的，建立一个低维的稠密向量空间
